# Documentação Homelab

Laboratórios focados em infraestrutura Linux, redes, VPN, DNS, sistemas embarcados e serviços web leves.

---

## Lab 1
Infraestrutura ZeroTier + VM Debian + TR1200. VPN overlay, roteamento e serviços NAS.

## Lab 2
Servidor DNS local utilizando dnsmasq em VM Debian integrado ao DHCP do OpenWrt.

## Lab 3
Port forwarding VPN utilizando ZeroTier e exposição remota de web server ESP32.

## Ferramentas Pandoc
Fluxo de conversão Markdown → HTML utilizando Pandoc e CSS GitHub Dark Mode.
